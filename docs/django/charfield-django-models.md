# 查菲尔德-姜戈车型

> 原文:[https://www.geeksforgeeks.org/charfield-django-models/](https://www.geeksforgeeks.org/charfield-django-models/)

CharField 是一个字符串字段，用于从小到大的字符串。它就像 C/C++中的字符串字段。CharField 通常用于存储名字、姓氏等小字符串。为了存储更大的文本，使用了文本字段。该字段的默认表单部件是[文本输入](https://docs.djangoproject.com/en/2.2/ref/forms/widgets/#django.forms.TextInput)。

**CharField 有一个额外的必需参数:**

```py
CharField.max_length
```

字段的最大长度(以字符为单位)。max_length 在数据库级别和 Django 的验证中使用 [MaxLengthValidator](https://docs.djangoproject.com/en/2.2/ref/validators/#django.core.validators.MaxLengthValidator) 强制执行。

**语法:**

```py
field_name = models.CharField(max_length=200, **options) 
```

## 如何使用 CharField？

CharField 用于在数据库中存储小型字符串。可以存储名、姓、地址详细信息等。应该给 CharField 一个参数 max_length 来指定它需要存储的最大字符串长度。在生产服务器中，部署了 Django 应用程序后，空间非常有限。因此，根据现场要求使用 max_length 总是最佳的。让我们创建一个我们创建的 CharField 的实例，并检查它是否工作。

## 字段选项

字段选项是为每个字段提供的参数，用于对特定字段应用某些约束或赋予特定特征。例如，将参数 null = True 添加到 CharField 将使其能够在关系数据库中存储该表的空值。
