# 工期现场-姜戈模型

> 原文:[https://www.geeksforgeeks.org/durationfield-django-models/](https://www.geeksforgeeks.org/durationfield-django-models/)

DurationField 是一个用于存储时间段的字段，在 Python 中由 [timedelta](https://www.geeksforgeeks.org/python-datetime-timedelta-function/) 建模。在 PostgreSQL 上使用时，使用的数据类型是间隔，在 Oracle 上，数据类型是 INTERVAL DAY(9)到 SECOND(6)。否则，将使用微秒的整数。DurationField 基本上存储一个持续时间，即两个日期或时间之间的差值。

**语法**

```py
field_name = models.DurationField(**options)
```

## Django 模型持续时间字段解释

因此，当您在项目上运行迁移时，会创建一个 geeks _ field**duration field**。它是一个存储 datetime.timedelta python 对象的字段。

## 如何使用 DurationField？

DurationField 用于在数据库中存储 python datetime.timedelta 实例。可以在数据库中存储基于时间或日期的任何类型的持续时间。要了解更多关于 datetime.timedelta 的信息，请查看 [Python | datetime.timedelta()函数](https://www.geeksforgeeks.org/python-datetime-timedelta-function/)。让我们尝试在上面创建的模型中存储一个 timedelta 实例。

## 蟒蛇 3

```py
# importing the model
# from geeks app
from geeks.models import GeeksModel

# importing datetime module
import datetime

# creating an instance of
# datetime.timedelta
d = datetime.timedelta(days =-1, seconds = 68400)

# creating an instance of
# GeeksModel
geek_object = GeeksModel.objects.create(geeks_field = d)
geek_object.save()
```

现在让我们在管理服务器中检查它。我们已经创建了一个极客模型的实例。
